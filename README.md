                                                                  "The Untold Project"

The Untold Project is a full-stack web application that I built completely on my own — from initial planning to final execution.

This project was not created by following a tutorial or a predefined guide. It began as a concept and stayed that way until a sudden wave of motivation hit me at 2 AM, pushing me to finally turn the idea into a fully working system.

This repository represents that execution.
About the Project
The Untold Project is designed and implemented as a real-world application rather than a demo or prototype.


I independently handled:

Planning and system architecture
Frontend development
Backend integration
Database design and management
Custom domain configuration
Hosting and deployment


Every part of the system was built, connected, and managed by me.
